Latex Tikz block-scheme

In this project I created the diagram for a research paper about behavioral patterns in education. It is a short project, however it illustrates various handy features of the Tikz environment, like canvas positioning of blocks, and a variety of parameters for arrows and alignment. It also illustrates the convenience of variable usage for large-scale diagrams.
